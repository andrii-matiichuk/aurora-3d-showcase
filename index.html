UX rules applied

Built on principles, not decoration

Every animation runs 150–400 ms, uses only transform/opacity, and respects prefers-reduced-motion.

Real-time WebGL engine

One WebGLRenderer per page, pixel ratio capped at ×2, InstancedMesh for repeated objects and delta-time via THREE.Clock — a steady 60 fps even on mid-range devices.

THREE.JS R128

Physics-based motion

GSAP spring curves and scrub timelines instead of linear transitions.

GSAP 3.12

Smooth orbit control

Click and drag to rotate the scene in spherical coordinates, with release inertia.

DRAG ORBIT

Accessible by default

Text contrast ≥4.5:1, an aria-label on the canvas, and full prefers-reduced-motion support with a reactive listener for OS setting changes.

WCAG AA

Design system in tokens

Semantic CSS variables, an 8px spacing scale, the Space Grotesk × DM Sans pairing and a Web3 palette — all from ui-ux-pro-max.
